  LENDING HEIGHTS TEAM DIRECTORY - INSTALLATION GUIDE (WINDOWS)
===============================================================================

üéâ Welcome! Let's get your Team Directory running in 3 easy steps!

===============================================================================
STEP 1: EXTRACT THE ZIP FILE
===============================================================================

1. Right-click the ZIP file
2. Select "Extract All..."
3. Choose a location (e.g., Documents\Projects\)
4. Click "Extract"

You should now have a folder called "teammate-directory-complete"

===============================================================================
STEP 2: OPEN IN VS CODE
===============================================================================

1. Open Visual Studio Code
2. Click "File" ‚Üí "Open Folder"
3. Navigate to and select "teammate-directory-complete"
4. Click "Select Folder"

===============================================================================
STEP 3: INSTALL & RUN
===============================================================================

1. Open the integrated terminal in VS Code:
   - Click "Terminal" ‚Üí "New Terminal"
   - OR press Ctrl + ` (backtick key)

2. Install dependencies (first time only):
   
   npm install
   
   This will take 1-2 minutes. Wait for it to complete.

3. Start the development server:
   
   npm run dev
   
4. Open your browser and go to:
   
   http://localhost:3000
   
   üéâ You should see your beautiful Team Directory!

===============================================================================
TROUBLESHOOTING
===============================================================================

‚ùå "npm is not recognized"
   ‚Üí You need to install Node.js first:
   ‚Üí Go to https://nodejs.org
   ‚Üí Download the LTS version (left button)
   ‚Üí Install it and restart VS Code

‚ùå Port 3000 already in use
   ‚Üí Try: npm run dev -- -p 3001
   ‚Üí Then open: http://localhost:3001

‚ùå Build errors about missing modules
   ‚Üí Delete node_modules folder
   ‚Üí Delete package-lock.json
   ‚Üí Run: npm install
   ‚Üí Run: npm run dev
